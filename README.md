# londonwald
Is London a forest? Answer the question using Python and Open Data. This repo has all the code and setup scripts so that you can
follow along the tutorial [published on codementor](https://www.codementor.io/blog/london-forest-gis-5tipxvoha5).

To get every component up and running, do:
```docker-compose up -d```
